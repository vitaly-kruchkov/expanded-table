# A data table with the ability to edit data.

---

## ⚙️ Local Development

### 1. Clone the repository

Prerequisites
• Node.js >= 20
• pnpm / npm / yarn

```bash
git https://github.com/vitaly-kruchkov/expanded-table/tree/main
cd expanded-table
```

### 2. Install dependencies

```bash
npm install
```

```bash
cd apps/api && npm install
```

### 3. Deployment

```bash
npm dev
```

### 4. Technologies

• React
• Next.js (for API)
• MUI
• Date fns
• Zustand

### 5. Main Functionality

## Editing cells - click on a cell to edit it

## Various data types:

• string
• number with support for integer and fractional values
• enum with a dropdown list
• Dates and times (timestamp)
• Arrays - tags
• Objects - JSON editor
